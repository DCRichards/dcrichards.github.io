AJAX

Stands for Asynchronous JavaScript and XML. A way of making asynchronous calls to a server and not having to replace a whole web page, only the parts we select.

                
var AJAXRequest = new XMLHttpRequest();
AJAXRequest.open("GET", "http://myserver.com/records/1", true);
AJAXRequest.send();

JavaScript Form Validation

Regular expressions can be used to validate forms. For this we use the onsubmit="" and use regular expressions in JS.

            
var checkForm = function(form) {
    var regularExp = /\(d{1,5})\s\d{1,6}/;
    return regularExp.test(form.phone.value);
};

< form id="myForm" onsubmit="return checkForm(this);">

JSP

Stands for Java Server Pages. Allows Java to be written inside HTML. The process for conversion is a follows:

  • The JSP Handler is invoked
  • If the page has been called before then exectute existing servlet
  • Build a servlet from the JSP page and execute

JSP Expressions are evaluated as a string

                
<%= date %>

JSP Scriptlets are included in the service() method

                
<% Drum lowKick = new Kick("lowKick", true); %>

JSP Declarations are included outside the service() method

                
<%! public int getValue() { return this.myVar; } %>

JSP Directives are used to specify properties of the servlet eg. extend a class

                
<%@ page import="java.util.*" extends="ArrayList" session="true" %>

JSP Actions are to specify an action within the page eg. a redirect

                
< jsp:forward page="myNewPage.jsp"/>

DOM

Document Object Model, we can access the DOM through Java with a few libraries. The basic process is to read in a file, make changes then write it out.

                
import org.w3c.dom.*;
import javax.xml.*;
import org.xml.*;

public class Example {

    private Document doc;

    public void readInFile() {
        DocumentBuilderFactory fact = DocumentBuilderFactory.newInstance();
        DocumentBuilder builder = fact.newDocumentBuilder();
        currentDoc = builder.parse(new File("myFile.xml")); 
    }
    
    public void modifyFile() {
        NodeList nodes = doc.getElementsByTagName("TAG");
        for (int i = 0; i < nodes.getLength() i++) {
            Node node = nodes.item(i);
            Text nodeText = (Text) node.getChildNodes().item(0);
            String content = nodeText.getNodeValue();
            Text newNodeText = doc.createTextNode(content + " NEW");
            RAMNode.replaceChild(newNodeText, nodeText);
    }
    
    public void writeFile() {
        //details of this don't matter
        FileWriter.writeFile(doc, "myFile.xml");
    }
    
    public static void main(String[] args) {
        Example ex = new Example();
        ex.readInFile();
        ex.modifyFile();
        ex.writeFile();
    }
}
